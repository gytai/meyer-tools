
### 功能
- [x] IM功能   
- [x] 个人Todo功能
- [x] 内部文件云盘
- [x] 文件预览以及分享   

### 客户端安装
* 1. npm install
* 2. 开发: npm run serve
* 3. 生产: npm run build

> 温馨提示： 部署前端使用反向代理和后端对接

### 服务端安装
* 1. npm install
* 2. pm2 start bin/www -name server
